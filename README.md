# My-Calculator

# Basic Calculator Web App

# Description
This is a basic Calculator web application built using HTML, CSS, and JavaScript. It performs standard arithmetic operations 
such as addition, subtraction, multiplication, and division. The project also includes features like clear, delete, percentage 
calculation, and a simple user-friendly interface. A dark mode toggle is included to enhance the user experience.

# Features
Perform basic arithmetic operations
Clear (C) and Delete (DEL) functionality
Percentage (%) calculation
Responsive  layout
Dark mode toggle
Simple and clean user interface

# Technologies Used
HTML ,
CSS ,
JavaScript
